# resamplr <img src="https://img.shields.io/badge/version-0.1.0-blue.svg" align="right"/>

`resamplr` is a pedagogically oriented R package implementing classical  
**resampling**, **randomization**, and **Monte Carlo** methods using only base R.  

It is designed for **graduate-level coursework** in Statistical Computing /  
Computational Methods (e.g., STAT 8670).  
The package aims to make algorithms *transparent* and *inspectable*, while  
providing mathematically correct implementations aligned with:

- **Rizzo (2019)** â€“ *Statistical Computing with R*  

- **Wegman & Solka (2011)** â€“ *Computational Statistics*  

- **Everitt & Hothorn (2006)** â€“ *A Handbook of Statistical Analyses Using R*  

The package is licensed under the **MIT License**, allowing open educational use.

## ðŸ“˜ Example Usage

1. Bootstrap CI for the Median

```{r}
set.seed(8670)
x <- rexp(50)            # Skewed, non-normal data

b <- boot_stat(x, stat = median, B = 2000)

boot_se(b)
boot_bias(b)
boot_ci(b, method = "percentile")
```

2. Two-Sample Permutation Test
```r
x <- rnorm(20, mean = 0)
y <- rnorm(20, mean = 0.5)

pt <- perm_test(x, y, B = 5000)

pt$stat_obs
pt$p_value
```

3. K-Fold Cross-Validation
```r
set.seed(1)
n <- 80
dat <- data.frame(x = rnorm(n))
y <- 1 + 2 * dat$x + rnorm(n)

model_fit <- function(data, y) lm(y ~ x, data = data)
predict_fun <- function(fit, data) predict(fit, newdata = data)
loss <- function(y_true, y_pred) mean((y_true - y_pred)^2)

cv_out <- kfold_cv(dat, y, model_fit, predict_fun, loss, K = 10)
cv_out$cv_estimate
```

4. Monte Carlo with Control Variates
```r
g  <- function(x) exp(x)
sampler <- function(n) rnorm(n)

h <- function(x) x     # Control variate
mu_h <- 0

set.seed(1)
plain <- mc_integrate(g, sampler, n = 10000, method = "plain")
ctrl  <- mc_integrate(g, sampler, n = 10000, method = "control",
                      control = list(h = h, mu_h = mu_h))

plain
ctrl
```
